⚡ How to use these scripts

1
Console: Open the game site, press F12 → Console tab, paste the script and hit Enter.
2
Tampermonkey: Install the Tampermonkey extension, click "Add new script", paste and save.
3
Tip: Console scripts run once per page load. Tampermonkey scripts auto-run every time you visit the site.
